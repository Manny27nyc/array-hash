Detailed Description

template<class GrowthFactor = std::ratio<3, 2>>
class tsl::ah::mod_growth_policy< GrowthFactor >

Grow the hash table by GrowthFactor::num / GrowthFactor::den and use a modulo to map a hash to a bucket. Slower but it can be usefull if you want a slower growth.
